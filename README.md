# Telegram Бот для парсинга мероприятий

Бот автоматически парсит новости о мероприятиях, конференциях, форумах и встречах стартаперов с популярных казахстанских сайтов.

## 🚀 Установка

### 1. Установите Python 3.8 или выше

Проверьте версию:
```bash
python --version
```

### 2. Установите зависимости

```bash
pip install -r requirements.txt
```

Или установите вручную:
```bash
pip install python-telegram-bot aiohttp beautifulsoup4 lxml
```

### 3. Получите токен бота

1. Откройте Telegram
2. Найдите @BotFather
3. Отправьте команду `/newbot`
4. Следуйте инструкциям и получите токен (например: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`)

### 4. Настройте бота

Откройте файл `bot.py` и найдите строку:
```python
BOT_TOKEN = "ВАШ_ТОКЕН_ЗДЕСЬ"
```

Замените `ВАШ_ТОКЕН_ЗДЕСЬ` на ваш токен от BotFather.

### 5. Запустите бота

```bash
python bot.py
```

## 📱 Использование

### Команды бота:

- `/start` - Начать работу с ботом
- `/events` - Показать актуальные мероприятия
- `/help` - Помощь

### Как работает бот:

1. Пользователь отправляет команду `/events`
2. Бот парсит новости с сайтов
3. Ищет упоминания о мероприятиях, конференциях, форумах
4. Возвращает список найденных событий с датами и ссылками

## 🔧 Настройка

### Добавление новых сайтов:

1. Откройте `bot.py`
2. Найдите список `URLS`
3. Добавьте новый URL в список

### Изменение ключевых слов для поиска:

В методе `parse_generic` измените список `keywords`:
```python
keywords = ['event', 'мероприятие', 'conference', 'конференция', ...]
```

1. Добавьте бота в канал/группу как администратора

## 🐛 Решение проблем

### Бот не запускается:
- Проверьте, правильно ли указан токен
- Убедитесь, что установлены все зависимости

### Бот не находит мероприятия:
- Некоторые сайты могут блокировать парсинг
- Попробуйте изменить User-Agent в коде
- Проверьте интернет-соединение

### Ошибки при парсинге:
- Структура сайтов может меняться
- Нужно обновить селекторы в коде парсера
